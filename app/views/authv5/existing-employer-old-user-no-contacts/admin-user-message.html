What does this mean?

Your employer does not have any contacts linked to their account.

As the first person to sign in to this account, you're automatically made an admin user.

Each employer must have at least one admin user.

What can an admin user do?

As an admin user you're able to:

  • add other users to the employer account
  • choose whether they have admin, standard or contact only access
  • delete any users you do not need
