What does this mean?

As the first person to sign in to this account, you're automatically made an admin contact.

Each employer must have at least one admin contact.

What can an admin contact do?

As an admin user you're able to:

  • add other contacts to the employer account
  • choose whether they have admin, standard or contact only access
